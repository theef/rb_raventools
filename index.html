rb_raventools

An unofficial ruby wrapper for the RavenTools SEO API. Requires an account with RavenTools and an API Key.

Using rb_raventools

Installation

Add this line to your application's Gemfile:

gem 'rb_raventools'

And then execute:

$ bundle

Or install it yourself as:

$ gem install rb_raventools

Setup

Once 'rb_raventools' is in your project, simply setup a RavenTools api client:

    client = RavenTools::Client.new("INSERT_YOUR_API_KEY")

and from there, you can perform method calls like:

    domains = client.domains

Methods

Domains

To find all domains connected to a particular account (as defined by the API Key), simply use:

    domains = client.domains

This method does not accept additional parameters or options.

Sample Response

Output will be an array of strings, containing all domains available.

    # domains = client.domains
    ["example1.com", "sub.example1.com", "example2.org"]

Profile Information

To retrieve available profile information:

    profile = client.profile_info

This method does not accept additional parameters or options.

Sample Response

Output will be a hash of values containing profile information (name, # of keywords).

    # profile = client.profile_info
    {"name"=>"Example Profile", "keyword_usage"=>"1000"}

Engines

To retrieve all available Search Engines:

    engines = client.engines

This method does not accept additional parameters or options.

Sample Response

Output will be an array of hash of values containing engine information (id, engine name).

    # engines = client.engines
    [{"id"=>"93", "name"=>"Yahoo! GR"}, {"id"=>"94", "name"=>"Bing NL"}]

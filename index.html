rb_raventools

Using rb_raventools

Installation

Add this line to your application's Gemfile:

gem 'rb_raventools'

And then execute:

$ bundle

Or install it yourself as:

$ gem install rb_raventools

Setup

Once 'rb_raventools' is in your project, simply setup a RavenTools api client:

client = RavenTools::Client.new("INSERT_YOUR_API_KEY")

and from there, you can perform method calls like:

domains = client.get_domains

or specify options / parameters such as:

keywords = client.get_keywords("DOMAIN_NAME", { format: "xml" })
